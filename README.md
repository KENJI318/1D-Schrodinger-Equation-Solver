# 1D-Schrodinger-Equation-Solver
A Schrödinger Equation solver that implements Finite Difference and Shooting methods to solve a general 1D piece-wise continuous potential $V(x)$.

When entering parameters, make sure they are entered in SI units only, including any distances or energies. When defining a potential, make sure that it is returned in $eV$, and the input $x$ is in $nm$.
The solvers output will be as follows: energies in $eV$, distances in $nm$, and $\psi$ in ${nm}^{-1/2}$.


All the files were created in Google Colab, and users are advised to do the same as the codes are written accordingly. 
Make sure to upload the .py files in MyDrive/Colab Notebooks file so that the notebooks are able to call the solver functions, or just change code for searching the location of these functions iin the notebook accordingly.
Run the notebooks and give it the required permission for the code to run smoothly.
